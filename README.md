# 一年级下册（第四至七单元）百词大赛通关练习系统说明

为了帮助班级高效组织**百词大赛**，我们基于第四到第七单元所有的生字词，全新设计了一套**多维互动练习系统**。它不仅在屏幕上具有极佳的视觉与发音交互，更对 A4 纸张打印进行了深度排版优化，支持一键打印空白试卷或标准答案。

2. **[原始生字源文件] [baici.html](file:///Users/louis/code/hanhan/yuwen/baici.html)**：您提供的原始生字词测验卷数据源。

## ✨ 核心特色功能

### 1. 四大精品关卡设计
* **第一关：百词听写大通关**：100% 覆盖原 `baici.html` 的 178 个字次，上方为标准四线三格拼音，下方为红色田字格，支持分单元听写或综合百词大赛。
* **第二关：同音形近字大侦探**：精选本阶段 8 组高频易错辨析（如 `在/再`、`像/象`、`近/进`、`凉/量`、`饱/抱`、`只/支`，以及经典的 `青/清/晴/情/请/睛` 青字族六兄弟填空）。
* **第三关：词语配对连连看**：设计 12 组由本阶段生字构成的核心词汇，打乱顺序排成两列，在屏幕上支持点击连线，打印时支持手写连线。
* **第四关：偏旁变身小魔术**：利用一年级重点的偏旁结构（如“青字族”和“包字族”），进行加偏旁组新字再组词练习。

### 2. 双模式一键切换
* **✏️ 学生练习模式**：汉字答案完全隐藏，田字格和横线全部空白。在屏幕上练习时，**点击田字格能单独浮现该汉字**（非常适合小孩子在屏幕上指读、描红或自我检测）。
* **🔍 教师批改模式**：一键显示所有标准答案。田字格内的答案会呈现为**淡红色描红字样**，同音字和偏旁题也会自动填充答案，便于家长或教师对照批改，也可用作“描红临摹本”。

### 3. 点击拼音播放发音 (🔊 Web Speech)
在网页中，小孩子可以**直接点击任意拼音四线格**，浏览器会自动调用系统语音，以标准的普通话、清晰且缓慢的语速朗读该汉字。即使没有家长在身边，孩子也能独立完成听写练习！

### 4. 完美适配 A4 纸张打印 (🖨️ Pro Print)
我们针对打印进行了深度的 CSS 媒体查询优化。当您点击“**一键打印 A4 练习卷**”时：
### 5. 🎲 一键随机乱序重组 (Shuffle Mode)
* 系统新增了“Fisher-Yates 随机打乱”机制。点击控制面板上的“🎲 一键乱序重组”，可将当前单元（或全单元综合大考）的所有生词顺序完全随机打乱！
* 这样能完美规避小孩子“依靠特定顺序或位置记忆”进行背诵，是真正的字词实力考察。
* 连线题（第三关）右列也将在点击时重新随机排序。

### 6. ⚙️ 智能自定义字词出题 (Custom Generator)
* 在控制面板上点击“⚙️ 自定义出题”按钮，即可从屏幕右侧平滑拉出一个优雅的**毛玻璃出题面板抽屉**（在 A4 打印模式下该面板会自动隐藏）。
* **超高容错智能提取**：您可以粘贴任何包含“拼音和汉字”的文本，例如：
  - `sī 思, chuáng 床, qián 前`
  - `sī(思) chuáng(床) qián(前)`
  - `[["sī","思"], ["chuáng","床"]]`
* 系统通过内置的正则配对提取器，在 0.1 秒内智能提取所有的拼音-汉字对，并将其重装载为全新试卷，所有点击发音和一键打印均可完美自适应运行！

## 🚀 快速使用指南

1. **直接双击打开**：
   在电脑或平板上，直接双击 **[baici_practice.html](file:///Users/louis/code/hanhan/yuwen/baici_practice.html)** 即可使用浏览器打开（推荐使用 Chrome, Safari, Edge 或 Firefox）。
   
2. **生成随机乱序新试卷**：
   * 选择需要听写的单元；
   * 点击控制台上的 **🎲 一键乱序重组**；
   * 此时界面听写网格将发生金光闪烁微动效，生词排列顺序瞬间洗牌打乱；
   * 直接开始作答或打印。

3. **自定义字词实时出题**：
   * 点击控制台上的 **⚙️ 自定义出题**；
   * 在右侧的抽屉输入区域内粘贴您的新词汇，点击 **✨ 立即解析生成**；
   * 试卷会瞬间自动重载，更新为您输入的全新词库。若要恢复初始数据，只需再次打开抽屉点击 **🔄 恢复默认字库** 即可。

4. **打印空白练习卷**：
   * 确保左上角处于 **学生练习模式**；
   * 在单元下拉菜单中选择“**综合百词大赛**”或特定的“**第四/五/六/七单元**”；
   * 点击右上角的“**一键打印 A4 练习卷**”（或在浏览器中按 `Cmd + P` / `Ctrl + P`）；
   * 在打印预览中，目标打印机选择“另存为 PDF”或直接输出到实体打印机。

5. **打印教师答案卷**：
   * 切换至 **教师批改模式**；
   * 此时网页上的所有答案将显现；
   * 点击“**一键打印 A4 练习卷**”即可输出带答案的批改卷。
